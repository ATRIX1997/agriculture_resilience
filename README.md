# Agriculture et résilience — Burkina Faso

Analyse exploratoire des données (EDA) sur les indicateurs agricoles et le développement rural au **Burkina Faso**, dans une perspective de compréhension des tendances et des enjeux de résilience du secteur.

## Contenu du dépôt

| Fichier / dossier | Description |
|-------------------|-------------|
| `EDA_Agriculture_Burkina_Faso.ipynb` | Notebook d’analyse (pandas, matplotlib, seaborn) |
| `agriculture_burkina.csv` | Données agricoles et ruralité (extrait / jeu utilisé dans le notebook) |
| `plots/` | Graphiques exportés depuis l’EDA |
| `sain_defis_1_by_abdel_kader_razack.pdf` | Document associé au projet |

## Source des données

Indicateurs issus de la **Banque mondiale**, diffusés via le **Humanitarian Data Exchange (HDX)** :

[Dataset HDX — World Bank Indicators for Burkina Faso](https://data.humdata.org/dataset/cd97d950-0124-468b-9bf0-71d232e47668/resource/81856343-2ffc-4336-8eb6-c1bd90ec9759/download/agriculture-and-rural-development_bfa.csv)

## Prérequis

- Python 3.10+ (recommandé)
- Bibliothèques : `pandas`, `matplotlib`, `seaborn`, `numpy`

```bash
pip install pandas matplotlib seaborn numpy
```

## Lancer l’analyse

```bash
jupyter notebook EDA_Agriculture_Burkina_Faso.ipynb
```

Ou avec JupyterLab / VS Code en ouvrant directement le notebook.
